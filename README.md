# Job-Applicant-Profile-System

This Python project is a **Candidate Profile Management System** using **inheritance**. It stores personal information, educational background, and work experience of candidates. The program displays complete professional profiles and determines eligibility for a **Senior Role (>5 years experience)** or a **Junior Role (≤5 years)**.

---

## 📂 Repository Files

- **`profile.py`** – The main Python program containing the implementation of candidate profile classes and methods.  
- **`output2.txt`** – Sample output generated by the program.  
- **`Output (2).png`** – Screenshot of the program output.
